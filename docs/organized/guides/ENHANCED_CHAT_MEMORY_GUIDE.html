üöÄ Overview

The Enhanced Chat Memory System revolutionizes conversation management in NIS Protocol v3. Instead of losing context between messages or sessions, the system now maintains a rich, persistent memory that includes:

  • Persistent Storage: All conversations stored permanently in SQLite
  • Semantic Search: Find relevant context from ANY past conversation
  • Topic Continuity: Automatic topic tracking and threading
  • Cross-Conversation Linking: Connect related discussions automatically
  • Enhanced Context: Up to 50 messages instead of just 8

üì° New API Endpoints

Memory Statistics

GET /memory/stats

Get comprehensive statistics about the memory system including storage size, conversation count, and performance metrics.

Conversation Search

GET /memory/conversations?query=&user_id=&limit=

Search conversations by content using semantic search or text matching.

Conversation Details

GET /memory/conversation/{conversation_id}

Get detailed analysis of a specific conversation including messages, topics, and semantic context.

Topic Discovery

GET /memory/topics

Get automatically discovered topics from conversations with importance scores.

Topic Conversations

GET /memory/topic/{topic_name}/conversations

Find all conversations related to a specific topic.

Memory Cleanup

POST /memory/cleanup?days_to_keep=90

Clean up old conversation data to manage storage space.

Context Preview

GET /memory/conversation/{conversation_id}/context?message=

Preview the context that would be used for a specific message in a conversation.

üîß Configuration

ChatMemoryConfig Settings

  • storage_path: "data/chat_memory/" - Storage location
  • max_recent_messages: 20 - Recent context limit
  • max_context_messages: 50 - Total context limit
  • semantic_search_threshold: 0.7 - Similarity threshold
  • enable_cross_conversation_linking: true - Link conversations
