## 🚀 Production Deployment

### Prerequisites

- Docker & Docker Compose
- 8GB+ RAM recommended
- Python 3.11+
- CUDA optional (for GPU acceleration)

### Deployment Steps

1. **Clone and checkout v4 branch**
```bash
git clone https://github.com/Organica-Ai-Solutions/NIS_Protocol.git
cd NIS_Protocol
git checkout v4.0-self-improving-consciousness
```

2. **Configure environment**
```bash
cp .env.example .env
# Edit .env with your API keys and config
```

3. **Start the system**
```bash
./start-cpu.sh  # or ./start-gpu.sh for GPU
```

4. **Verify all phases initialized**
```bash
docker-compose -f docker-compose.cpu.yml logs backend | grep "Phase"
```

Expected output:
```
🧬 Phase 1: Evolutionary consciousness initialized
🤖 Phase 8: Physical embodiment initialized
🔍 Phase 9: Consciousness debugger initialized
🔬 Phase 10: Meta-evolution initialized
✅ All v4.0 consciousness phases initialized
```

5. **Test basic functionality**
```bash
curl http://localhost:8000/v4/consciousness/debug/explain | jq '.'
```

### Monitoring

- **Logs**: `docker-compose -f docker-compose.cpu.yml logs -f backend`
- **Health**: `curl http://localhost:8000/infrastructure/status`
- **Metrics**: `curl http://localhost:8000/metrics`
