NIS Protocol Examples

Practical implementations of the Neuro-Inspired System Protocol

Overview

This section contains practical examples of the NIS Protocol in action. Each example demonstrates how the protocol's components work together to create adaptive, context-aware systems for different application domains.

These examples are designed to be educational starting points that you can adapt for your own applications. The complete source code for all examples is available in the examples directory of the GitHub repository.

The toll booth example demonstrates the NIS Protocol's layered architecture in action

Featured Example: Automated Toll Booth System

Toll Booth Simulation

The automated toll booth example demonstrates a complete implementation of the NIS Protocol for processing vehicles at a toll plaza. The system uses multiple agents working together to identify vehicles, verify payment methods, detect anomalies, and manage toll operations.

Key Components:

  • Vision Agent: Processes camera feeds to identify vehicle type, license plate, and occupancy
  • RFID Agent: Detects and processes transponder information for registered vehicles
  • Memory Agent: Maintains records of vehicle history, payment status, and known issues
  • Emotion Agent: Modulates system behavior based on traffic conditions and anomaly detection
  • Reasoning Agent: Makes decisions about toll amounts, lane assignments, and security flags
  • Action Agent: Controls barrier gates, lane signals, and customer information displays

NIS Protocol Features Demonstrated:

  • Multi-modal input processing (visual, RFID, database)
  • Context-aware decision making with memory integration
  • Emotional state modulation for adaptive security thresholds
  • Learning from historical patterns to optimize traffic flow
  • Coordinated agent communication across cognitive layers

Example Code Snippet:

from nis_protocol.core import NISRegistry
from nis_protocol.agents import VisionAgent, RFIDAgent, MemoryAgent, EmotionAgent, ReasoningAgent, ActionAgent
from nis_protocol.emotion import EmotionalStateSystem

# Initialize the NIS Protocol registry
registry = NISRegistry()

# Set up the emotional state system
emotional_system = EmotionalStateSystem()
registry.set_emotional_state(emotional_system)

# Create and register agents
vision_agent = VisionAgent("vision_1", "Processes camera feeds")
rfid_agent = RFIDAgent("rfid_1", "Processes transponder signals")
memory_agent = MemoryAgent("memory_1", "Maintains vehicle records")
emotion_agent = EmotionAgent("emotion_1", "Modulates system behavior")
reasoning_agent = ReasoningAgent("reasoning_1", "Makes toll decisions")
action_agent = ActionAgent("action_1", "Controls toll booth equipment")

# Process a vehicle approaching the toll booth
def process_vehicle(camera_image, rfid_signal):
    # Initial emotional state - standard monitoring
    emotional_state = emotional_system.get_state()
    
    # Step 1: Vision agent processes the vehicle image
    vision_result = vision_agent.process({
        "payload": {"camera_image": camera_image},
        "emotional_state": emotional_state
    })
    
    # Step 2: RFID agent processes transponder signal
    rfid_result = rfid_agent.process({
        "payload": {"rfid_signal": rfid_signal},
        "emotional_state": emotional_state
    })
    
    # Step 3: Check vehicle history in memory
    vehicle_id = vision_result["payload"]["license_plate"] or rfid_result["payload"]["transponder_id"]
    memory_result = memory_agent.process({
        "payload": {
            "operation": "retrieve",
            "key": vehicle_id
        },
        "emotional_state": emotional_state
    })
    
    # Step 4: Update emotional state based on vehicle history
    if memory_result["payload"].get("payment_issues"):
        emotional_system.update_dimension("suspicion", 0.3)
    if memory_result["payload"].get("flagged"):
        emotional_system.update_dimension("suspicion", 0.5)
        
    # Step 5: Reasoning agent makes decision
    reasoning_result = reasoning_agent.process({
        "payload": {
            "vision_data": vision_result["payload"],
            "rfid_data": rfid_result["payload"],
            "memory_data": memory_result["payload"]
        },
        "emotional_state": emotional_system.get_state()
    })
    
    # Step 6: Action agent executes the decision
    action_result = action_agent.process(reasoning_result)
    
    return action_result
