Getting Started with NIS Protocol

A guide to implementing the Neuro-Inspired System Protocol in your projects

Installation

The NIS Protocol v3.0 is available as a Python package from TestPyPI: 

pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple/ nis-protocol-v3

Note: This installs from TestPyPI. For production use, the package will be available on PyPI with pip install nis-protocol-v3

Alternatively, you can clone the repository directly from GitHub: 

git clone https://github.com/Organica-Ai-Solutions/NIS_Protocol.git
cd NIS_Protocol
pip install -e .

Basic Usage

Here's a simple example of creating an NIS Protocol agent: 

from nis_protocol import Agent, EmotionalState

# Initialize the agent with default emotional state parameters
agent = Agent(
    name="assistant",
    emotional_state=EmotionalState()
)

# Define perception function
def process_input(input_data):
    # Process incoming data
    perception_result = agent.perceive(input_data)
    
    # Interpret the perception
    interpretation = agent.interpret(perception_result)
    
    # Reason about the interpretation
    reasoning_result = agent.reason(interpretation)

    return action

Configuring the Emotional State

You can customize the emotional state parameters when creating an agent: 

from nis_protocol import Agent, EmotionalState

# Create a custom emotional state
emotional_state = EmotionalState(
    suspicion=0.3,  # Slightly cautious
    urgency=0.7,    # Quite urgent
    confidence=0.5, # Neutral confidence
    interest=0.6,   # Somewhat interested
    novelty=0.5     # Neutral novelty
)

# Initialize agent with custom emotional state
agent = Agent(
    name="security_monitor",
    emotional_state=emotional_state
)
